# Lotties in html

Prueba para conocer como funciona lotties con un html basico

## Installation

No requiere instalacion, solo pull

```bash
Sin instalacion
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
